El Reto:

Crear un nuevo repositorio en su computador

Repositorios

Un Repositorio es básicamente un proyecto. Usted puede imaginarlo como una carpeta de proyecto con todos los archivos relacionados dentro de ella. De hecho, esto es lo que verá en su computador de todas formas.

Usted le dice a Git lo que es su proyecto y Git empezará el seguimiento de todos los cambios en esa carpeta. Los archivos agregados, eliminados o incluso el cambio a una simple letra en un archivo. -- Todo esto es seguido por Git, además de esto les agrega una marca de tiempo. Esto es un control de versiones.

Usando la terminal

Terminal (or Bash) es una manera de usar su computador usando comandos. Usted puede renombrar archivos, abrir archivos, crear nuevas carpetas y moverse entre directorios (carpetas), todo escribiendo comandos. Usted puede incluso ejecutar un editor de texto (como Vim) en su terminal y nunca tiene que salir de ella!

Además de navegar en su computador, usted tambien puede usar programas en la terminal los cuales tengan una Interfaz de línea de comandos [command-line interface (CLI)], esto significa que ellos pueden ser ejecutados con comandos en la terminal. Git-it es uno de ellos, usted esta usando la terminar para ejecutarlo!. Git es otro ejemplo. Usted puede acceder y controlar Git a través de comandos en la terminal, tal como usted lo hará prontamente!

En Git-it usted aprenderá algunas acciones básicas de la línea de comandos las cuales serán descritas en los siguientes pasos.

Primer paso: Crear un repositorio

Usted debe crear una nueva carpeta e inicializarla como un repositorio de Git.

Para hacer esto más fácil, nombre su carpeta igual que el proyecto, qué tal 'hola-mundo'.

Usted puede escribir estos comandos uno a la vez en su terminal.

Para crear una nueva carpeta:

$ mkdir hola-mundo

Para entrar a esa carpeta:

$ cd hola-mundo

Para crear una nueva instancia de git para el proyecto:

$ git init

Eso es todo! Esto va a retornarle una nueva línea. Si usted quiere asegurarse nuevamente de que es un repositorio de git, escriba git status y si esto no retorna 'fatal: Not a git repository...', usted es oro!

Verifique ejecutando

git-it verify

Pase al siguiente reto ejecutando

git-it

Ayuda!

  • Cree una nueva carpeta (directorio)
    • $ mkdir <FOLDERNAME>
  • Entrar a una carpeta (cambiar directorio)
    • $ cd <FOLDERNAME>
  • Listar todos los elementos en una carpeta
    • $ ls
  • Inicializar git en una carpeta
    • $ git init
