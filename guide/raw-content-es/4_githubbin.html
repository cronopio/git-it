Crear una cuenta en GitHub, añadir el nombre de usuario a su configuración de git.

Mejor juntos

El repositorio que usted ha creado está sólo en su computador, lo cual es útil, pero hace que sea muy difícil compartirlo y trabajar con otros en su proyecto. No se preocupe, para eso está hecho GitHub!

Primer Paso: Crear una cuenta en GitHub

GitHub es un sitio web que permite a las personas subir las cosas en las cuales ellas están trabajando con git y así trabajar juntos fácilmente.

Visite github.com y presione 'sign up' para crear una cuenta gratis. Bien hecho, bienvenido!

Segundo Paso: Agregar nombre de usuario a Git

Agregue su nombre de usuario a su configuración de Git, la cual será necesaria pra verificar los retos posteriores. Guardelo exactamente como usted lo creó en GitHub — Use las mayúsculas igual que como lo hizo en GitHub., y recuerde que usted no necesita escribir los "<" y ">" .

Agregar su nombre de usuario en su configuración:

$ git config --global user.username <USerNamE>

Verifique ejecutando

git-it verify

Pase al siguiente reto

git-it

¿ No pasa ?

El nombre de usuario en GitHub & Git no coinciden

Un error común es que su nombre de usuario en GitHub no coincide con el que usted configuró usando git config. Por ejemplo, 'JLord' no es lo mismo que'jlord'

Para correguir el nombre de usario, simplemente utilice el comando que usted ejecutó anteriormente, pero con la capitalización correcta:

$ git config --global user.username <USerNamE>

Después de que haya hecho las actualizaciones, verifique de nuevo!
