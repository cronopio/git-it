Git

Git es software de código abierto (libre para usar por cualquiera) escrito por Linus Torvalds quien también escribió el sistema operativo Linux.

Es un programa para dar seguimiento a los cambios hechos al código durante el tiempo, conocido como control de versiones.

Primer paso: Instalar Git

  • Windows: Es recomendable descargar GitHub para Windows, que incluye Git como un instalador fácil: windows.github.com. Utilice Git Shell como si fuera la terminal.
  • Mac: También puede descargar GitHub para Mac, que incluye Git, mac.github.com (desde Preferences, seleccione instalar desde la línea de comandos), o
    • descargue Git directamente desde: git-scm.com/downloads y siga las instrucciones de instalación.

Git no es como otros programas en su computador. Usted no verá ningún icono en el escritorio, pero estará siempre disponible para usted y usted podrá acceder a Git en cualquier momento desde su terminal o línea de comandos (en la cual ya esta usando Git-it!) o desde aplicaciónes de escritorio como Github para Mac o Windows

Segundo paso: Configurar Git

Una vez este instalado, abre la terminal (Bash, Shell, Consola, etc). Puede verificar que esta realmente alli, escribiendo:

$ git --version

Esto debe mostrar la version de Git que esta usando y debe parecer algo como:

git version 1.9.1

(Cualquier version por encima de 1.7.10 está bien.)

Siguiente, configurar Git para que sepa con quien asociar los cambios:

Ponga su nombre:

$ git config --global user.name "<Su Nombre>"

Ahora su correo:

$ git config --global user.email "<sucorreo@ejemplo.com>"

Verifique ejecutando

git-it verify

Pasa al siguiente reto ejecutando git-it

Ayuda!

Los signos de pesos $ que aparecen frecuentemente en documentación de programación, significan que esa linea es un comando para ejecutar en la linea de comandos (mira los comandos de arriba). Usted no tiene que escribir el simbolo $, sólo escriba lo que viene despues. Por ejemplo, para ejecutar el comando de verificar, usted escribirá git-it verify en la linea de comandos
