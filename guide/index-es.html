Git-it

Bienvenido a Git-it!

Aquí te daremos las instrucciones paso a paso para cada uno de los retos en el Taller Git-it. Mira la versión en Chino Tradicional y en Inglés.

Esta guía viene incluida en Git-it cuando se instala, por lo tanto funciona desconectado de Internet y puedes usarla como recurso útil en cualquier momento (mira las instrucciones cuando selecciones un reto en la terminal). Diviértete!

Retos Git-it

  1. Obten Git Instalar y configurar Git
  2. Repositorio Crea un repositorio local
  3. Commit Verifica el estado, agrega y confirma los cambios
  4. GitHubbin Crea una cuenta de GitHub
  5. Remote Control Conecta repositorios locales a repositorios remotos en GitHub.com
  6. Forks y Clones Clona y crea un Fork de un repositorio de código abierto
  7. Las ramas (branchs) no son sólo para aves Crea una rama para tus propios cambios
  8. Es un mundo pequeño Agrega y sincroniza un colaborador (Hola @reporobot!)
  9. Pull, Nunca desactualizado Empuja (Push) y hala (Pull) tus cambios para sincronizar con GitHub.com
  10. Me halas por favor Crea una petición para que tus cambios sean halados por otros (pull request)
  11. Combinaciones Combina y borra ramas
