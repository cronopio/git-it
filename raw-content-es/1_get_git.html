El Reto:

Instalar Git en su computador y configurar el nombre y el correo.

Git

Git es software de código abierto (libre para usar por cualquiera) escrito por Linus Torvalds quien tambien escribio el sistema operativo Linux.

Es un programa para dar seguimiento a los cambios hechos al codigo durante el tiempo, conocido como control de versiones.

Primer paso: Instalar Git

  • Windows: Es recomendable descargar GitHub para Windows, que incluye Git como un instalador fácil: windows.github.com. Utilice Git Shell como si fuera la terminal.
  • Mac: Tambien puede descargar GitHub para Mac, que incluye Git, mac.github.com (desde Preferences, seleccione instalar desde la linea de comandos), o
    • descargue Git directamente desde: git-scm.com/downloads y siga las instrucciones de instalación.

Git no es como otros programa en su computador. Usted no verá ningun icono en el escritorio, pero estará siempre disponible para usted y usted podrá acceder a Git en cualquier momento desde su terminal o linea de comandos (en la cual ya esta usando Git-it!) o desde aplicaciónes de escritorio como Github para Mac o Windows

Segundo paso: Configurar Git

Once it is installed, open terminal (aka Bash, aka Shell, aka Prompt). You can verify that it's really there by typing:

$ git --version

This will return the version of Git that you're running and look something like this:

git version 1.9.1

(Any version 1.7.10 or higher is fine.)

Next, configure Git so it knows who to associate your changes to:

Set your name:

$ git config --global user.name "<Your Name>"

Now set your email:

$ git config --global user.email "<youremail@example.com>"

Verify with

git-it verify

Go to the next challenge

git-it

Tips

Dollar signs $ are often used in programming documentation to signify that the line is command line code (see the code snippets above). You don't actually type the $ in though, only type what comes after. For instance, to run the verify command above, you're actually going to type git-it verify into terminal.
